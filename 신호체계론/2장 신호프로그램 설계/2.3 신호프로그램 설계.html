2.3 신호프로그램 구성

2.3.1 현시 구분

2.3.1.1 개요

  • 교통류 간의 상충관계를 파악해야 한다.
    • 상충이 없는 교통류 : 하나의 현시로 구현
    • 상충 교통류 : 현시를 분리해 구현
  • 부분적 상충
    • 대향 방향 교통류
    • 평행하게 통행하는 교통류(보행자, 자전거 등)

2.3.1.2 좌회전 교통류

  • 신호체계에 의해 보호되는 교통류
  • 교통안전 측면 등 다음의 요인을 고려
    • 대향 교통류의 속도가 높을 경우
    • 좌회전 차량의 통행이 원활하게 이루어져야 할 경우
    • 좌회전 교통량이 많거나 상충 교통류의 교통량이 많을 경우
    • 조건적으로 상충되는 교통류와의 시인성이 불량할 경우
    • 다수의 상충으로 좌회전 교통류의 주의력이 필요한 경우
      • 노면전차
      • 마주보는 우회전
      • 동시에 현시가 부여되는 평행 교통류
  • 2차선 이상이 배정된 경우 신호로 보호되어야 한다.

2.3.1.3 우회전 교통류

  • 교통섬이 없는 교차로 진입로에서 일반적으로 신호등을 설치하지 않는다.
  • 다음의 경우 설치할 수 있다.
    • 도로 측면 교통량이 많은 경우
    • 별도 현시를 부여하는 것이 효율적인 경우
  • 3단 신호등 우회전 교통신호가 필요한 경우
    • 2차로로 우회전
    • 원활한 곡선반경이 필요
    • 시인성 불량
    • 도로 측면 교통량이 많은 경우
  • 대향 교통의 좌회전 교통류와 현시를 구분해야 한다.

2.3.1.4 노면전차와 노선버스

  • intergreen time을 고려한 특수현시 적용

2.3.1.5 보행 교통

  • 기본 상태
    • 녹색 : 차량
    • 적색 : 보행자
  • 신호요구 발생 시
    • 차량
      • 녹-황-적
      • 보행 현시 종료 후
      • 적/황-녹
    • 보행자
      • 적-녹-적
  • 다차선 교통류의 경우 보행자는 신호적으로 분리된다.
  • 다음의 경우 분리 신호가 적용된다.
    • 회전 교통류가 원활하게 통행해야 한다.(교통량이 많거나 상충빈도가 높다.)
    • 차량과 보행자간의 시인성이 불량
    • 보행 교통량이 많다.
    • 좌회전 교통류가 대향 교통류로부터 차두간격을 찾기가 어려울 경우

2.3.1.6 자전거 교통

  • 차량 신호등을 준수
  • 기본 형태
    • 차량과 공동으로 신호운영
    • 자전거 전용 신호체계
    • 보행자와 자전거 공동신호

2.3.2 현시의 수

  • 모든 상충 교통류를 신호적으로 분리할 경우
    • 3지 교차로 : 3개의 현시
    • 4지 교차로 : 4개의 현시
  • 2현시
    • 조건적으로 상충되는 교통류가 보호받지 못하게 된다.
    • 일부 회전 교통류들은 같은 현시에 통행이 된다.
    • 교통 소통 측면에서 일반적으로 효율적으로 평가받는다.
  • 신호주기 측면에서 ingergreen time 최소화해야 한다.
  • 녹색시간의 길이가 유사한 교통류들은 같은 현시로 묶어야 한다.
